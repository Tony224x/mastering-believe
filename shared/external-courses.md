# Reférences cours universitaires — IA / ML (gratuits sur YouTube)

Index public des cours d'IA / Machine Learning publiés librement sur YouTube par les universités de premier rang. Ce fichier sert de **référence partagée** pour les domaines de ce repo : quand un module théorique s'appuie sur un de ces cours, on **référence directement la vidéo YouTube** ci-dessous.

## Attribution & usage

Les contenus pédagogiques (cours, vidéos) listés ci-dessous sont la **propriété des universités** et de leurs auteurs respectifs (Stanford, MIT, CMU, UC Berkeley, Harvard). Ce repo :

- **ne redistribue aucun contenu** — les liens pointent exclusivement vers les vidéos YouTube officielles publiées par les universités
- **ne reproduit pas mot-à-mot** de longs passages des cours — les modules théoriques de ce repo paraphrasent et synthétisent les notions, et renvoient à la vidéo source pour le contenu exhaustif
- **n'est pas affilié** à ces universités ni endossé par elles — il s'agit d'un repo communautaire d'apprentissage qui s'appuie sur ces ressources publiques au même titre qu'un étudiant qui prend des notes
- **n'a aucun usage commercial** — apprentissage personnel et open-source, sans revente

Tout lecteur intéressé est encouragé à **regarder les vidéos directement sur YouTube** — c'est l'expérience voulue par les universités et ça soutient leur audience.

Si un titulaire de droits considère qu'un module de ce repo cite trop largement un de ses contenus, ouvrir une issue : la portion concernée sera réécrite ou retirée.

## Comment référencer un cours dans un module théorique

Quand une notion théorique du repo est traitée par une lecture, ajouter une ligne en bas du module :

```markdown
## Pour aller plus loin

- **Stanford CS336 — Lecture 3 : Tokenization** → [YouTube](https://www.youtube.com/watch?v=...)
- Voir aussi : [`shared/external-courses.md`](../../../shared/external-courses.md)
```

Le lien YouTube pointe **directement** vers la vidéo concernée (pas la playlist), pour que le lecteur arrive sur le bon segment.
