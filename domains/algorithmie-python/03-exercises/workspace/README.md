# Workspace — Algorithmie Python

Espace pour ecrire **tes** solutions aux exercices.

> **Ce dossier est ignore par git** (sauf `README.md` et `.gitkeep`). Tout ce que tu y mets reste local — tu peux l'utiliser sans craindre de polluer un fork ou une PR.

## Organisation

```
workspace/
├── 01-easy/      # Tes solutions aux exercices faciles
├── 02-medium/    # Tes solutions aux exercices intermediaires
├── 03-hard/      # Tes solutions aux exercices difficiles
```

## Workflow recommande

1. Lis l'enonce dans `../01-easy/01-two-sum.md` (ou autre)
2. Cree ta solution ici : `01-easy/01-two-sum.py`
3. Execute : `python domains/algorithmie-python/03-exercises/workspace/01-easy/01-two-sum.py`
4. Une fois ta solution validee, compare avec celle du dossier `../solutions/`

## Convention de nommage

Garde le meme nom que l'exercice original (`01-two-sum.md` → `01-two-sum.py`). Ca rend la comparaison avec la solution plus simple.
