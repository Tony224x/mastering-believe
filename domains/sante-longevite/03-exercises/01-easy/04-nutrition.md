# Exercices — Module 04 : Nutrition fondee sur les preuves

> **Prerequis** : avoir lu `01-theory/04-nutrition.md`
> **Objectif des exercices** : appliquer les invariants alimentaires a des situations concretes, construire une assiette type selon les principes du module, et pratiquer la lecture critique d'une etude nutritionnelle.

---

## Exercice 1 — Planifier une journee de repas "invariants-first"

### Objectif

Construire une journee de repas qui respecte les invariants alimentaires identifies dans le module (aliments peu transformes, fibres, proteines adequates, sucres ajoutes limites) — sans imposer un "regime" particulier.

### Consigne

Imaginez que vous planifiez vos repas pour demain. En vous appuyant **uniquement sur les invariants du module** (pas de recette imposee, pas de schema mediterraneen obligatoire) :

1. Proposez un menu pour les 3 repas principaux + 1 collation eventuelle.
2. Pour chaque repas, identifiez quel(s) invariant(s) il respecte.
3. Identifiez un aliment ultra-transforme que vous pourriez remplacer dans votre alimentation habituelle, et proposez une alternative.

**Format attendu :** tableau ou liste structuree, court (1 page max).

### Criteres de reussite

- [ ] Chaque repas inclut une source de fibres (legumes, fruits, legumineuses ou cereales completes)
- [ ] Chaque repas inclut une source de proteines (animale ou vegetale — pas de prescription)
- [ ] Au moins 2 repas n'utilisent pas d'aliment ultra-transforme comme composant principal
- [ ] L'apprenant a identifie au moins 1 substitution d'ultra-transforme avec une alternative
- [ ] Aucune etiquette de regime n'est appliquee ("c'est du keto", "c'est vegan", etc.)

## Exercice 2 — Decrypter un titre nutritionnel avec la lentille du module 01

### Objectif

Appliquer la methode de lecture critique (module 01 : niveaux de preuve, correlation vs causalite) a un titre de presse sante typique.

### Consigne

Voici trois titres fictifs inspires de titres reels :

> **Titre A** : "Manger du chocolat noir reduit le risque cardiaque de 22 % (etude sur 100 000 personnes)"
> **Titre B** : "Un essai clinique randomise confirme que les acides gras omega-3 abaissent les triglycerides"
> **Titre C** : "Les personnes qui sautent le petit-dejeuner ont 30 % de risque de diabete en plus"

Pour chaque titre, repondez en 3-5 phrases :
1. De quel type d'etude s'agit-il probablement (observationnelle ou ECR) ?
2. Peut-on conclure a une relation causale ? Pourquoi ?
3. Quel facteur de confusion potentiel pourrait expliquer le resultat ?

### Criteres de reussite

- [ ] Le type d'etude est correctement identifie pour les 3 titres
- [ ] La distinction "correlation ne signifie pas causalite" est appliquee pour au moins 2 titres
- [ ] Au moins 1 facteur de confusion plausible est nomme par titre
- [ ] La reponse pour le Titre B reconnait que les ECR ont un statut de preuve superieur

## Exercice 3 — Evaluer son propre rapport aux "guerres de regimes"

### Objectif

Prendre conscience de ses propres biais sur les regimes alimentaires, et les confronter aux invariants du module.

### Consigne

1. Listez 2-3 croyances que vous aviez (ou que vous entendez autour de vous) sur un regime specifique (ex. "le gluten est mauvais pour tout le monde", "les graisses font grossir", "le keto est le meilleur pour perdre du poids").
2. Pour chaque croyance, cherchez son "invariant sous-jacent" : y a-t-il un principe soutenu par la preuve qui en constitue le noyau raisonnable ?
3. Reformulez chaque croyance dans le langage des invariants.

**Exemple** :
- Croyance : "les glucides font grossir"
- Invariant sous-jacent : les sucres ajoutes et les ultra-transformes riches en glucides simples contribuent a la suralimentation passive
- Reformulation : "reduire les sucres ajoutes et les ultra-transformes est soutenu par la preuve ; les glucides complexes (cereales completes, legumineuses) font partie des invariants positifs"

### Criteres de reussite

- [ ] 2-3 croyances identifiees, y compris celles que l'apprenant lui-meme portait
- [ ] Chaque croyance est confrontee a un ou plusieurs invariants du module
- [ ] La reformulation est factuelle, sans condamner la croyance initiale de facon moralisatrice
- [ ] L'apprenant reconnait la difference entre une croyance populaire et une preuve scientifique
