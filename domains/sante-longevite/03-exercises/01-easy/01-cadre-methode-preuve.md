## Exercice 1 — Classer des sources par niveau de preuve

### Objectif
Appliquer la hierarchie des preuves a des sources reelles pour developper le reflexe de lecture critique.

### Consigne

Vous trouverez ci-dessous **5 affirmations sante** avec leur source. Pour chacune :

1. Identifiez le **type d'etude** (ECR, meta-analyse d'ECR, cohorte prospective, etude de cas, opinion d'expert, autre).
2. Attribuez-lui un **niveau de preuve** (1 = plus fort a 5 = plus faible, selon la hierarchie du module 01).
3. Indiquez si l'affirmation **peut** ou **ne peut pas** etre interpretee comme causale.

---

**Affirmation A** :
"Dans une meta-analyse Cochrane de 30 ECR, le regime mediterraneen est associe a une reduction moderee du risque d'AVC en prevention primaire." *(Rees K et al., 2019, Cochrane Database Syst Rev)*

**Affirmation B** :
"Dans une etude sur 10 000 infirmieres suivies pendant 20 ans, celles qui consomment plus de 2 cafes par jour ont 18 % moins de risque de diabete de type 2 que celles qui n'en consomment pas."

**Affirmation C** :
"Mon docteur m'a dit que boire du jus de citron le matin stimule le metabolisme."

**Affirmation D** :
"Dans un ECR de 3234 sujets prediabetiques (DPP, NEJM 2002), une intervention intensive sur le mode de vie (alimentation + exercice) a reduit l'incidence du diabete de type 2 de 58 % sur 2,8 ans." *(Knowler WC et al., 2002)*

**Affirmation E** :
"Une etude sur 1200 personnes en bonne sante a montre que ceux qui dorment moins de 6h ont 34 % plus de rhumes apres exposition au rhinovirus que ceux qui dorment ≥ 7h." *(Cohen S et al., 2015, Sleep)*

---

### Criteres de reussite

- [ ] Chaque affirmation a un type d'etude identifie correctement (ou approche correcte)
- [ ] Chaque affirmation a un niveau de preuve attribue (1 a 5) avec une justification en 1-2 phrases
- [ ] Vous avez correctement identifie laquelle ou lesquelles permettent une **inference causale** (et pourquoi les autres ne le permettent pas)
- [ ] Vous avez note au moins une **limite** de l'affirmation la plus forte (A ou D)

## Exercice 2 — Analyser un titre de presse

### Objectif
Dechiffrer un titre sante alarmiste ou enthousiaste et aller chercher l'information de qualite derriere.

### Consigne

Lisez ce titre fictif (style presse grand public) :

> **"Les personnes qui mangent du chocolat noir chaque jour vivent 5 ans de plus — une etude le prouve"**

L'article cite une etude observationnelle menee en Suede sur 8 ans, sur 45 000 participants. Les amateurs de chocolat noir ≥ 20g/jour avaient une mortalite toutes causes reduite de 11 % par rapport aux non-consommateurs (p < 0,05).

Repondez aux questions suivantes par ecrit (une reponse par question) :

1. Quel est le **type d'etude** ? Peut-on conclure que le chocolat *cause* une vie plus longue ?
2. Le resultat est "statistiquement significatif" (p < 0,05). Est-ce suffisant pour conclure ? Que manque-t-il dans l'article ?
3. Citez **au moins deux facteurs confondants** plausibles qui pourraient expliquer l'association.
4. Quel type d'etude serait necessaire pour etablir la causalite ? Decrivez-en brievement le design.
5. Le titre dit "une etude *le prouve*" — reformulez le titre de maniere honnete et calibree.

### Criteres de reussite

- [ ] Le type d'etude est identifie correctement et l'impossibilite de conclure causalement est argumentee
- [ ] Vous avez nomme l'intervalle de confiance et/ou la taille d'effet comme elements manquants
- [ ] Au moins 2 facteurs confondants pertinents sont proposes avec une explication courte
- [ ] Le design ECR est decrit avec au moins : randomisation, groupe controle, mesure du critere principal
- [ ] Le titre reformule est factuel, ne pretend pas a la causalite, et mentionne le niveau de preuve

## Exercice 3 — Cartographier votre noyau Pareto personnel

### Objectif
Appliquer le cadre du module 01 a votre propre situation — non pour vous juger, mais pour identifier les leviers les plus actionnables.

### Consigne

Sur une feuille (ou dans votre espace `03-exercises/workspace/`), creez un tableau personnel avec les 6 comportements du noyau Pareto :

| Comportement | Etat actuel (1-5) | Principale friction |
|---|---|---|
| Sommeil ≥ 7 h (regularite) | | |
| Activite aerobique (150 min/sem) | | |
| Renforcement musculaire (≥ 2j/sem) | | |
| Alimentation peu transformee | | |
| Tabac (non-fumeur = 5) | | |
| Lien social & alcool (moderation) | | |

Instructions :
- **Etat actuel 1-5** : 1 = tres insuffisant, 5 = solide et regulier
- **Principale friction** : une phrase sur ce qui vous empeche d'atteindre 4-5 sur ce levier (pas d'explication longue — soyez factuel)
- Identifiez le **levier ou le gain marginal serait le plus grand** pour vous (pas necessairement celui ou vous etes le plus faible — le plus actionnable)
- Formez **une hypothese** : "Si j'ameliore [levier X] de [delta concret], je m'attends a [effet observable]." Exemple : "Si je dors 30 min de plus en me couchant a 22h30 au lieu de 23h, je m'attends a moins de fatigue en milieu de matinee."

### Criteres de reussite

- [ ] Les 6 comportements sont evalues avec une note 1-5 et une friction en 1 phrase
- [ ] Un levier prioritaire est identifie avec une justification courte (pas juste "c'est le plus mauvais")
- [ ] Une hypothese en format "Si... alors..." est formulee, avec un effet observable concret et mesurable
- [ ] Aucune promesse de resultat — l'hypothese est formulee comme une **experience a tester**, pas une certitude

> **Note** : Cet exercice reste dans votre `workspace/` — il n'a pas vocation a etre commite dans le depot.
