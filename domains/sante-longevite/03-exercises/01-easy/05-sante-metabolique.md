> **Objectif des exercices** : interpreter les biomarqueurs metaboliques de base dans un cadre educatif, comprendre l'impact du mode de vie sur la glycemie, et appliquer les enseignements du DPP a un cas concret.

## Exercice 1 — Interpreter un bilan metabolique fictif (usage educatif uniquement)

### Objectif

Apprendre a lire les principaux biomarqueurs metaboliques dans un cadre strictement educatif — pas pour se diagnostiquer, mais pour comprendre ce que votre medecin regarde.

### Consigne

> **⚠️ Rappel** : cet exercice est educatif. Ne l'utilisez pas pour interpreter votre propre bilan sans l'avis de votre medecin.

Voici le bilan fictif de "Marie", 45 ans, sedentaire, alimentation riche en ultra-transformes :

| Biomarqueur | Valeur Marie | Reference indicative (ADA / OMS) |
|-------------|-------------|----------------------------------|
| Glycemie a jeun | 1,10 g/L | < 1,00 normal ; 1,00-1,25 prediabete ; ≥ 1,26 diabete |
| HbA1c | 5,9 % | < 5,7 normal ; 5,7-6,4 prediabete ; ≥ 6,5 diabete |
| Triglycerides | 1,80 g/L | < 1,50 g/L souhaitable |
| HDL-cholesterol | 0,38 g/L | > 0,50 g/L (femme) |
| Tour de taille | 88 cm | > 80 cm = risque eleve (femme, OMS) |

1. Quels biomarqueurs sont en dehors des valeurs de reference indicatives ?
2. Sur la base des criteres du syndrome metabolique (module 05, section 5), Marie remplit-elle les criteres ? Combien ?
3. Quels leviers du mode de vie sont les plus directement associes a une amelioration de ces biomarqueurs, selon le module ?
4. Que manque-t-il pour conclure sur la sante de Marie ? (raisonnez sur les limites de cet exercice)

### Criteres de reussite

- [ ] Les 4-5 biomarqueurs hors reference sont correctement identifies
- [ ] La conclusion sur le syndrome metabolique cite le bon nombre de criteres (≥ 3)
- [ ] Au moins 2 leviers de mode de vie sont nommes avec leur mecanisme d'action (ex. activite physique → sensibilite a l'insuline)
- [ ] L'apprenant reconnait qu'un medecin est indispensable pour une interpretation clinique complete
- [ ] Aucun "diagnostic" n'est pose — le champ lexical reste educatif

## Exercice 2 — Appliquer le raisonnement du DPP a un scenario

### Objectif

Comprendre l'ampleur de l'effet du mode de vie sur le risque de diabete de type 2 en travaillant le chiffre cle du DPP.

### Consigne

"Paul" a 50 ans, glycemie a jeun a 1,08 g/L (zone prediabete indicative), BMI 28, sedentaire. Son medecin lui explique qu'il a un risque augmente de developper un diabete de type 2.

1. En vous basant sur les resultats du DPP (Knowler et al., NEJM 2002), quelle reduction de risque Paul pourrait-il esperer si il atteignait les objectifs de l'intervention mode de vie (-7 % de poids + 150 min/sem d'activite) ?
2. Comparez cet effet a celui de la metformine dans le meme essai.
3. Que signifie une reduction de 58 % en termes absolus ? Si 100 personnes dans la situation de Paul ne font rien et 58 developpent un DT2 sur 3 ans (chiffre simplifie), combien seraient evites avec l'intervention mode de vie ?
4. Pourquoi la metformine hors diabete est-elle presentee comme "recherche" dans ce domaine et non comme une recommandation ?

### Criteres de reussite

- [ ] La reduction de 58 % (IC95 % 48-66) est correctement citee avec sa source
- [ ] La comparaison avec la metformine (-31 %) est effectuee
- [ ] La conversion risque relatif → risque absolu (question 3) est tentee, avec une explication du raisonnement
- [ ] Le statut de la metformine hors diabete (essai TAME non finalise = recherche) est correctement rappele

## Exercice 3 — Cartographier les connexions entre les leviers et la sante metabolique

### Objectif

Visualiser comment les 6 leviers du noyau Pareto convergent tous sur la sante metabolique.

### Consigne

Dessinez (a la main ou dans un outil simple) une carte mentale ou un tableau avec **"Sante metabolique"** au centre. Pour chaque levier du noyau (sommeil, activite physique, nutrition, stress, lien social, tabac/alcool) :

1. Tracez une fleche vers "Sante metabolique"
2. Sur la fleche, notez le mecanisme principal (ex. : "Activite physique → augmente la captation musculaire du glucose → ameliore la sensibilite a l'insuline")
3. Identifiez pour chaque levier : est-ce qu'un mauvais etat de ce levier *degrade* la sante metabolique ? Et inversement, son amelioration *beneficie-t-elle* a la sante metabolique ?

### Criteres de reussite

- [ ] Les 6 leviers sont representes
- [ ] Chaque fleche comporte un mecanisme specifique (pas juste "c'est bon pour la sante")
- [ ] Au moins 4 leviers ont un mecanisme correctement nomme (ex. : stress → cortisol → glycemie elevee ; sommeil insuffisant → insulinoresistance)
- [ ] L'apprenant identifie que les leviers sont interdependants (ameliorer l'un aide les autres)
