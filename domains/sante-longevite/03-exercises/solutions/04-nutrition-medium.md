## Exercice 1 — Auditer des regimes etiquetes via les invariants

### Corrige modele

**Menu "keto" A** :
- Peu transforme : **viole** (bacon industriel, fromage fondu, saucisses = ultra-transformes).
- Fibres : **viole** (quasi pas de legumes).
- Proteines : respecte (mais sources transformees).
- Sucres ajoutes/trans : risque trans/additifs des charcuteries. → **Globalement non conforme.**

**Menu "keto" B** :
- Peu transforme : respecte. Fibres : respecte (epinards, salade, brocolis). Proteines : respecte (oeufs, saumon, tofu/poulet). Sucres/trans : OK. → **Conforme** malgre l'etiquette identique a A.

**Menu "vegan" C** :
- Peu transforme : **viole** (cereales sucrees, nuggets vegetaux industriels, pizza industrielle). Fibres : faibles malgre l'etiquette vegetale. Sucres ajoutes : eleves (cereales sucrees). → **Non conforme** bien que "vegan".

**2. L'etiquette predit-elle la qualite ?**
Non. Keto A (mauvais) et keto B (bon) portent la **meme etiquette** mais ont une qualite opposee. Vegan C montre qu'une etiquette percue comme "saine" peut etre ultra-transformee. C'est exactement la regle du module : **la structure (invariants) compte plus que l'etiquette.**

**3. Reformulations (etiquette conservee)**
- Keto A → remplacer bacon/fromage fondu/saucisses par oeufs + avocat + legumes verts + huile d'olive (keto conforme).
- Vegan C → remplacer cereales sucrees par flocons d'avoine + fruits ; nuggets industriels par legumineuses/tofu + legumes (vegan conforme).

**Enseignement cle** : auditer un regime = verifier les invariants, pas le nom. "Keto bien formule" et "vegan bien formule" peuvent etre tous deux sains ; mal formules, tous deux mediocres.

## Exercice 2 — Lire PREDIMED avec ses reserves

### Corrige modele

**1. Generalisabilite (validite externe)**
Les sujets sont a **haut risque CV** (Espagne). Un effet etabli chez eux ne se transfere pas automatiquement a une personne jeune et en bonne sante : c'est la **validite externe** (a qui le resultat s'applique-t-il vraiment ?). On peut raisonnablement penser que les invariants restent benefiques, mais l'ampleur du -30 % n'est pas garantie hors de la population etudiee.

**2. Relatif vs absolu selon le risque de base**
Un "-30 % relatif" applique a un risque de base eleve evite beaucoup de cas en **absolu** ; applique a un risque de base faible (personne jeune saine), le meme -30 % evite tres peu de cas. C'est le meme principe qu'au module 01 : l'effet absolu depend du risque de depart. PREDIMED, mene sur haut-risque, "rentabilise" mieux l'effet relatif.

**3. Presentation honnete de PREDIMED**
"PREDIMED est un ECR avec un **critere cardiovasculaire dur** (force notable), montrant ~30 % de reduction du critere composite chez des sujets a haut risque. **Reserves** : la publication 2013 a ete retractee pour irregularites de randomisation, republiee en 2018 avec effets comparables ; et la revue Cochrane 2019 conclut a une preuve **moderee** pour les AVC, plus limitee pour la mortalite. C'est une preuve solide mais a presenter avec ces nuances."

**4. Ce que PREDIMED autorise / n'autorise pas**
- Autorise : dire que le schema mediterraneen est **un exemple de pattern alimentaire soutenu par un ECR** avec un critere CV dur, chez une population a risque.
- N'autorise PAS : dire que c'est **LE seul** regime valide, ni qu'il est superieur pour tous, ni qu'il "prouve" un benefice equivalent chez les personnes a bas risque. D'autres patterns partageant les memes invariants peuvent atteindre des objectifs similaires.

**Enseignement cle** : un ECR fort se lit avec sa population, son risque de base et ses reserves. "C'est un ECR" ne dispense pas de demander "chez qui, avec quel effet absolu, et avec quelles failles connues ?".

## Exercice 3 — Assiette Harvard pour 3 contextes

### Corrige modele

**Contexte 1 — petit budget**
- Matin : flocons d'avoine + banane (fibres, glucides complets) ; oeuf (proteine).
- Midi : riz complet + lentilles + carottes/oignons surgeles (50 % legumes, 25 % cereales, 25 % legumineuses).
- Soir : pommes de terre + chou + oeufs ou conserve de sardines rincee.
- Fibres : avoine, lentilles, legumes. Proteines : oeufs, lentilles, sardines.
- Piege : plats prepares "pas chers" ultra-transformes → alternative : legumineuses seches/conserves + surgeles bruts.

**Contexte 2 — peu de temps**
- Matin : yaourt nature + fruits surgeles + flocons.
- Midi : batch de quinoa + legumes surgeles non panes saisis + conserve de pois chiches rincee.
- Soir : poele rapide tofu/poulet + melange de legumes surgeles + huile d'olive.
- Fibres : quinoa, pois chiches, legumes. Proteines : yaourt, pois chiches, tofu/poulet.
- Piege : surgeles **panes/sauces** ou plats micro-ondables ultra-transformes → alternative : surgeles "bruts" (legumes nature, poisson non pane).

**Contexte 3 — sans viande**
- Matin : tartines completes + houmous + tomates.
- Midi : bowl riz complet + tofu + edamame + legumes varies.
- Soir : dahl de lentilles + epinards + riz complet (associer legumineuses + cereales pour les acides amines).
- Fibres : cereales completes, legumineuses, legumes. Proteines : tofu, edamame, lentilles, houmous.
- Piege : "faux-mues" vegetaux industriels (nuggets/steaks ultra-transformes) → alternative : tofu/tempeh/legumineuses.

**Criteres d'une bonne reponse** :
- Proportions ~50/25/25 respectees, 3 journees reellement differenciees par contrainte.
- Surgeles/conserves **bruts** correctement classes comme acceptables (≠ ultra-transformes).
- Un piege ultra-transforme + alternative par contexte.

**Enseignement cle** : les invariants sont compatibles avec tous les budgets et emplois du temps. Le surgele/conserve brut est un allie ; le piege n'est pas "transforme" mais "ultra-transforme".
