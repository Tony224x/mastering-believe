# Exercices — Module 07 : Capstone — La Boîte à Outils du Jugement

## Exercice 1 — Assembler sa checklist de pré-décision

### Objectif
Adapter la checklist du module 07 à une décision concrète personnelle.

### Consigne
Choisissez une décision réelle et neutre à prendre dans les 2 prochaines semaines. Remplissez la checklist complète :

```
DÉCISION : ___________  |  DATE : ___________

1. CLARIFIER
   Décision exacte : ___________
   Options réelles : ___________
   Horizon de temps : ___________

2. BIAIS
   Ancrage ? Estimation indépendante : ___________
   Disponibilité ? Taux de base : ___________
   Cadrage ? Reformulation inverse : ___________
   Confirmation ? Argument contre l'option favorite : ___________

3. PROBABILITÉS
   Scénario principal : _____%  |  Classe de référence : ___________

4. CONSÉQUENCES
   Optimiste / Central / Pessimiste + probabilités
   Scénario ruineux : ___________

5. VÉRIFICATION
   Fait clé 1 vérifié via : ___________
   Fait clé 2 vérifié via : ___________

6. DÉCISION & SUIVI
   Décision : ___________  |  Probabilité succès : _____%  |  Date revue : ___________
```

### Critères de réussite
- [ ] Décision réelle, neutre et concrète.
- [ ] Toutes les sections remplies.
- [ ] Au moins 2 biais explicitement examinés (pas juste cochés).
- [ ] Probabilité chiffrée avec classe de référence.
- [ ] Date de revue fixée.

## Exercice 2 — Construire son journal de prévisions (1 mois)

### Objectif
Mettre en place un journal de prévisions opérationnel et calculer le score de Brier initial.

### Consigne
**Phase 1** : formulez 10 prédictions sur des événements neutres dans les 4 semaines suivantes.

Format : `| Date | Question binaire | p (%) | Date résolution | Outcome | (p−o)² | Note |`

Règles : questions binaires, dates fixes, probabilités numériques, classes de référence notées.

**Phase 2** (après résolutions) : remplir Outcome et (p−o)².

**Phase 3** (analyse) :
1. Score de Brier moyen.
2. Courbe de calibration mentale : vos prédictions à 70-80 % se réalisent-elles 70-80 % du temps ?
3. Zone de sur-/sous-confiance identifiée ?

### Critères de réussite
- [ ] 10 prédictions avec toutes les colonnes.
- [ ] Questions binaires et non ambiguës.
- [ ] Classes de référence pour au moins 7 prédictions.
- [ ] (Phase 2) Score de Brier calculé.
- [ ] (Phase 3) Biais identifié ou score commenté.

## Exercice 3 — Appliquer le protocole complet sur une décision + vérification

### Objectif
Combiner les 3 outils (checklist, journal, protocole de vérification) sur un cas concret.

### Consigne
Choisissez une décision qui implique des informations externes. Appliquez :

**Outil 1 — Checklist (version courte, 6 questions)** :
1. Décision exacte ?
2. Biais vérifié en premier ? Résultat ?
3. Probabilité estimée + classe de référence ?
4. Scénario ruineux ?
5. Information à vérifier ?
6. Décision + date de revue ?

**Outil 2 — Journal** : enregistrer une prédiction liée à cette décision.

**Outil 3 — Vérification** : une information clé → appliquer SIFT complet et documenter les étapes + résultat.

**Livrable** : document de synthèse (1 page) résumant les 3 outils appliqués.

### Critères de réussite
- [ ] Les 3 outils utilisés sur le même cas.
- [ ] Checklist : au moins un biais examiné.
- [ ] Journal : prédiction binaire avec probabilité et date de résolution.
- [ ] Vérification : étapes concrètes documentées avec résultat.
- [ ] Synthèse lisible par quelqu'un d'autre.
