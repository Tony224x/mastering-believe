# PLAN — Communication & Persuasion

> Curriculum fige. Ne pas modifier sans revue collective.
> Agent A : modules 01-03 + fichiers racine. Agent B : modules 04-07.

---

## Principe directeur

**Persuasion ethique uniquement.** Chaque module rappelle ou applique le filtre CTR (Consentement / Transparence / Reciprocite honnete). Les techniques puissantes sont enseignees d'abord comme *defense* (reperer quand elles sont utilisees contre soi), puis comme outil — uniquement quand le fond est vrai.

---

## Garde-fou ethique (rappel permanent)

```
CTR — avant toute tentative de persuasion :
  C · Consentement  — l'autre peut dire non sans piege
  T · Transparence  — intentions + faits materiels visibles
  R · Reciprocite   — benefice mutuel reel vise
```

**Hors-scope absolu** : manipulation coercitive, dark patterns, PUA, spin politique, propagande, fausse rarete/autorite, pression abusive, exemples politiquement clivants.

---

## Statut de la preuve (a signaler dans les modules concernes)

| Sujet | Statut empirique |
|-------|-----------------|
| Trio ethos/pathos/logos (Aristote) | Cadre conceptuel solide, confirme par la SEP et l'ELM (O'Keefe) |
| Storytelling / transportation narrative | Base solide (meta-analyses Van Laer 2014, Shen 2015) — conditionnel au canal |
| Principes Cialdini | Reels mais **effets modestes** — Many Labs 2 (2018) : ~54 % repliques, taille d'effet reduite |
| Pre-Suasion / priming (Cialdini 2016) | **Plus fragile** — plusieurs effets de social priming non repliques |
| CNV (Rosenberg) | **Base de preuve mince** — cadre pratique utile, pas d'intervention "prouvee" |
| Voss (*Never Split the Difference*) | **Experience terrain** (FBI), zero essai controle — complementaire a Fisher & Ury |
| Negociation raisonnee (Fisher & Ury) | **Solide** — issu du Harvard Negotiation Project (1979), cursus universitaires |

## Curriculum fige — 7 modules

### Module 01 — Fondations & ethique de la persuasion
**Slug** : `01-fondations-ethique`
**Duree** : ~45 min
**Objectif** : Poser le cadre conceptuel (ethos/pathos/logos) et la charte ethique CTR ; distinguer persuasion rationnelle et manipulation.
**Sources** : Aristote / Kennedy, SEP Aristotle's Rhetoric, SEP Ethics of Manipulation, O'Keefe (Persuasion: Theory and Research).
**Encart ethique** : Difference entre persuasion (preserve l'autonomie) et manipulation (la contourne) — SEP "Ethics of Manipulation".

### Module 02 — Ecouter d'abord
**Slug** : `02-ecouter-dabord`
**Duree** : ~45 min
**Objectif** : Maitriser l'ecoute active et le modele OSBD (CNV) ; savoir refleter et etiqueter les emotions sans evaluer.
**Sources** : Rosenberg (NVC, 3e ed. 2015), Voss & Raz (Never Split the Difference, 2016).
**Signal epistémique** : CNV — cadre pratique utile, base de preuve heterogene (scoping review PMC 2024) ; a ne pas survendre.

### Module 03 — Clarte & ecriture qui se comprend
**Slug** : `03-clarte-ecriture`
**Duree** : ~45 min
**Objectif** : Diagnostiquer et corriger les opacites d'un texte ; appliquer les principes personnages-acteurs/verbes-actions et concision.
**Sources** : Pinker (The Sense of Style, 2014), Williams & Bizup (Style: Lessons in Clarity and Grace, ed. courantes).

### Module 04 — Messages qui collent
**Slug** : `04-messages-qui-collent`
**Duree** : ~45 min
**Objectif** : Appliquer le modele SUCCESs (Heath) pour rendre une idee memorable et transmissible.
**Sources** : Heath & Heath (Made to Stick, 2007), meta-analyses Van Laer 2014 et Shen 2015.

### Module 05 — Les principes de persuasion (et comment y resister)
**Slug** : `05-principes-persuasion`
**Duree** : ~45 min
**Objectif** : Connaitre les 7 principes de Cialdini, evaluer leur statut de replication, les utiliser honnetement et s'en defendre.
**Sources** : Cialdini (Influence, 2021), Pre-Suasion (2016), Many Labs 2 (Klein et al., 2018).
**Signal epistémique** : Many Labs 2 — ~54 % repliques, effets plus petits ; enseigner comme tendances, pas leviers mecaniques.

### Module 06 — Negociation raisonnee
**Slug** : `06-negociation`
**Duree** : ~45 min
**Objectif** : Conduire une negociation gagnant-gagnant : interets vs positions, options a benefice mutuel, criteres objectifs, BATNA ; complementer avec l'empathie tactique.
**Sources** : Fisher, Ury & Patton (Getting to Yes, 2011), Voss (Never Split the Difference, 2016), PON Harvard, MOOC Michigan Siedel.

### Module 07 — Capstone : convaincre honnetement
**Slug** : `07-capstone-convaincre-honnetement`
**Duree** : ~60 min (capstone)
**Objectif** : Produire un pitch oral (2 min), un ecrit persuasif (1 page) et conduire une simulation de negociation — le tout evalue sur clarte, ethique CTR et resultat mutuel.
**Sources** : Synthese des 6 modules ; TED Guide (Anderson, 2016) pour le pitch.

## Conventions de contenu (rappel)

- Langue : **francais** pour toute la theorie et les exercices.
- Exemples : **non clivants** (travail, sante, vie quotidienne, projets). Jamais politiques.
- Concret avant abstrait : chaque module ouvre sur un avant/apres de message reel.
- Flash-cards : 4-5 par module de theorie (Q&A spaced-repetition).
- Exercices : 3 progressifs (objectif / consigne / criteres) + solution modelisee.
