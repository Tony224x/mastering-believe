# Exercices (medium) — Module 01 : Fondations & ethique de la persuasion

> **Prerequis** : avoir lu `01-theory/01-fondations-ethique.md` et fait les exercices `01-easy/01-fondations-ethique.md`
> **Duree estimee** : 40-50 min
> **Solutions** : `03-exercises/solutions/01-fondations-ethique-medium.md`

> **Garde-fou ethique (charte CTR)** : tous les messages que vous produisez ici doivent passer le filtre **Consentement / Transparence / Reciprocite honnete**. Le but n'est jamais de "faire dire oui" mais de rendre une demande legitime claire et complete. Si un message ne convainc qu'en cachant un fait ou en pressant l'autre, il est a refuser.

---

## Exercice 1 — Equilibrer le trio quand il penche trop d'un cote

### Objectif

Apprendre a *reequilibrer* un message persuasif reel qui s'appuie trop sur une seule dimension (souvent pathos ou logos), sans tomber dans l'exces inverse.

### Consigne

Pour chacun des deux messages ci-dessous :

1. Identifiez la dimension **dominante** et celle(s) **manquante(s) ou faible(s)**.
2. Diagnostiquez le **risque concret** que cre le desequilibre (ex. : un logos seul "passe au-dessus" de l'auditoire ; un pathos seul declenche la mefiance).
3. Reecrivez le message (5-7 phrases) en reequilibrant les trois dimensions **sans gonfler artificiellement** la dimension faible.
4. Annotez votre version : [E] ethos, [P] pathos, [L] logos.

**Message A** — Email d'une responsable d'equipe pour faire adopter un nouvel outil de suivi de projet :
> "Cet outil est utilise par 78 % des equipes du secteur, il reduit le temps de reunion de 23 %, il s'integre a notre messagerie via API REST, et son cout est amorti en 4,2 mois selon notre simulation."

**Message B** — Message d'un membre d'association pour mobiliser des benevoles :
> "On a vraiment besoin de vous, c'est important pour le quartier, ce serait tellement triste de devoir annuler. S'il vous plait, repondez vite, on compte sur vous, ca nous ferait chaud au coeur."

### Criteres de reussite

- [ ] La dimension dominante et les dimensions faibles sont nommees pour chaque message.
- [ ] Le risque du desequilibre est explicite (pas "il manque quelque chose" mais *quel effet* sur le destinataire).
- [ ] La version reecrite contient les trois dimensions, annotees.
- [ ] La dimension renforcee s'appuie sur des elements reels (pas un pathos invente ni un faux ethos).
- [ ] Le message B ne perd pas sa chaleur authentique mais gagne un logos verifiable (combien de benevoles ? pour quoi ? quand ?).

## Exercice 2 — Le test du "miroir d'information"

### Objectif

S'entrainer a appliquer le test central du module ("Si j'etais a la place de l'autre et que je savais tout ce que je sais, accepterais-je ?") pour reperer une manipulation par **omission**, le cas le plus subtil.

### Consigne

Lisez les trois scenarios. Chaque message est *factuellement vrai* mais cache potentiellement un fait pertinent. Pour chacun :

1. Identifiez le **fait omis** qui changerait la decision de l'autre s'il le connaissait.
2. Appliquez le test du miroir d'information : la decision de l'autre serait-elle differente avec ce fait ? Justifiez.
3. Concluez : persuasion ethique ou manipulation par omission (cf. SEP / Springer 2023, "omettre une preuve pertinente = manipulation") ?
4. Reecrivez en version transparente.

**Scenario 1** — Vous vendez votre velo d'occasion :
> "Velo en tres bon etat, peu utilise, freins recents, ideal pour la ville." (vous savez que le cadre a une micro-fissure reparee a la soudure que vous ne mentionnez pas)

**Scenario 2** — Vous recommandez un restaurant a un collegue :
> "Ce restaurant est excellent, j'y vais souvent." (vous omettez que votre cousin en est le proprietaire)

**Scenario 3** — Vous proposez a votre equipe un creneau de reunion :
> "Je propose vendredi 16h, c'est le creneau qui arrange le plus de monde d'apres le sondage." (le sondage donnait 16h *et* mardi 10h a egalite, mais 16h vous arrange personnellement)

### Criteres de reussite

- [ ] Le fait omis est identifie precisement pour chaque scenario.
- [ ] Le test du miroir est applique explicitement (la decision changerait-elle ? oui/non + pourquoi).
- [ ] La distinction omission anodine / omission manipulatoire est argumentee (toute omission n'est pas une manipulation — seule celle d'un fait *pertinent pour la decision*).
- [ ] Les trois versions transparentes integrent le fait omis sans s'auto-flageller (transparence != confession dramatisee).

## Exercice 3 — Construire un argumentaire complet sous contrainte CTR

### Objectif

Produire un message persuasif de longueur moyenne (180-220 mots) qui mobilise le trio complet *et* anticipe une objection legitime — le tout en restant strictement dans le perimetre ethique.

### Consigne

Choisissez **une** situation parmi les suivantes :

**Situation A** — Vous proposez a votre manager de passer une partie de votre equipe en semaine de 4 jours en test.

**Situation B** — Vous proposez a la copropriete de votre immeuble d'installer un local a velos a la place d'un espace de stockage peu utilise.

**Situation C** — Vous proposez a un club sportif amateur de creer un creneau "debutants" le samedi matin.

Redigez un message qui :
1. Mobilise ethos, pathos, logos (annotes [E]/[P]/[L]).
2. Contient au moins **un fait verifiable** (chiffre, source, observation datable — vous pouvez utiliser un chiffre plausible en le signalant comme "a verifier").
3. **Anticipe une objection** de l'autre partie et y repond honnetement (pas en la balayant).
4. Se termine par une demande concrete laissant le "non" possible.

Puis verifiez le filtre CTR : une ligne par garde-fou (C / T / R).

### Criteres de reussite

- [ ] Les trois dimensions sont presentes et annotees.
- [ ] L'objection anticipee est reelle (pas un homme de paille facile a abattre).
- [ ] La reponse a l'objection ne cache aucun cout pour l'autre.
- [ ] La demande finale est concrete et refusable sans piege.
- [ ] Le filtre CTR est verifie en trois lignes explicites.
- [ ] Le message fait 180-220 mots.
