# Workspace — Neural Networks & LLMs

Espace pour ecrire **tes** solutions aux exercices.

> **Ce dossier est ignore par git** (sauf `README.md` et `.gitkeep`). Tout ce que tu y mets reste local — code, notebooks, figures, checkpoints.

## Organisation

```
workspace/
├── 01-easy/      # Tes solutions aux exercices faciles
├── 02-medium/    # Tes solutions aux exercices intermediaires
├── 03-hard/      # Tes solutions aux exercices difficiles
```

## Workflow recommande

1. Lis l'enonce dans `../01-easy/<exercice>.md` (ou autre)
2. Cree ta solution ici : `01-easy/<exercice>.py` ou `.ipynb`
3. Execute : `python domains/neural-networks-llm/03-exercises/workspace/01-easy/<fichier>.py`
4. Compare avec `../solutions/` une fois termine

## Convention de nommage

Garde le meme nom que l'exercice original. Ca rend la comparaison avec la solution plus simple.
