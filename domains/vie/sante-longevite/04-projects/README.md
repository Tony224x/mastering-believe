# Gabarit — Plan sante personnel (template portfolio)

> **Objectif :** Ce gabarit vous permet de formaliser votre plan sante personnel issu du capstone (module 07) dans un format structuré, exportable et utilisable comme reference de suivi sur le long terme.
>
> **Usage :** Copiez ce fichier dans votre espace personnel (hors du repo), remplissez chaque section, et partagez-le avec votre medecin pour validation.

---

> **⚠️ Disclaimer medical.** Ce plan est un **outil educatif personnel**. Il ne constitue pas une prescription medicale. Toute decision de modification du mode de vie (exercice, alimentation, complements, medicaments) doit etre discutee avec un professionnel de sante. Ce document est un point de depart pour une conversation medicale, pas un substitut.
