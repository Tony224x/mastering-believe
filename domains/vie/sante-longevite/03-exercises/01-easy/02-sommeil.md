## Exercice 1 — Auto-evaluer son sommeil avec un outil valide (ISI)

### Objectif
Utiliser l'Index de Severite de l'Insomnie (ISI), un outil valide cliniquement, pour obtenir une mesure objective de la qualite de votre sommeil — pas un ressenti vague.

### Consigne

L'**Insomnia Severity Index (ISI)** est un questionnaire en 7 items valide dans la litterature clinique (Morin CM et al., *Sleep*, 2011). Voici sa version française adaptee.

Pour chaque question, notez de **0 a 4** selon l'echelle ci-dessous, en referant aux **2 dernieres semaines** :

**Echelle** : 0 = Aucun | 1 = Leger | 2 = Modere | 3 = Severe | 4 = Tres severe

| # | Question | Note (0-4) |
|---|----------|-----------|
| 1 | Difficulte a vous endormir | |
| 2 | Difficulte a rester endormi(e) (reveils nocturnes) | |
| 3 | Reveil trop tot le matin | |
| 4 | Satisfaction de votre sommeil actuel (0 = tres satisfait, 4 = tres insatisfait) | |
| 5 | Dans quelle mesure les problemes de sommeil interferent avec votre fonctionnement diurne (fatigue, concentration, humeur, etc.) | |
| 6 | Dans quelle mesure ces problemes sont-ils *remarques* par les autres (qualite de vie, rendement) | |
| 7 | Dans quelle mesure etes-vous preoccupe(e) par vos problemes de sommeil actuels | |

**Calcul du score** : Additionnez les 7 items.

| Score total | Interpretation |
|------------|----------------|
| 0-7 | Pas d'insomnie cliniquement significative |
| 8-14 | Insomnie sous-clinique (legere) |
| 15-21 | Insomnie clinique moderee |
| 22-28 | Insomnie clinique severe |

Complementez avec **une semaine de journal de sommeil** (simple) : pour chaque nuit, notez en 1 ligne :
- Heure de coucher / heure de lever
- Duree estimee du sommeil effectif (heures)
- Qualite subjective (1-5)
- Evenement notable (cafeine, alcool, stress, ecran tardif)

### Criteres de reussite

- [ ] Le questionnaire ISI est complete avec un score calcule et son interpretation notee
- [ ] Un journal de sommeil est tenu au minimum **5 nuits** consecutives
- [ ] Vous avez identifie **au moins une variable** (cafeine, heure de coucher, alcool, ecrans) correle avec les nuits de moins bonne qualite
- [ ] Si le score ISI est ≥ 15, vous avez note l'intention de consulter un medecin ou un therapeute forme a la CBT-I (pas de prescription d'auto-traitement ici)

## Exercice 2 — Diagnostiquer ses perturbateurs de sommeil

### Objectif
Identifier les 2-3 perturbateurs principaux de votre sommeil en appliquant la grille d'hygiene vue dans le module 02, et formuler un plan d'action minimaliste.

### Consigne

Evaluez chaque levier d'hygiene du sommeil selon votre habitude actuelle :

| Levier | Habitude actuelle (libre) | Impacte votre sommeil ? (O/N/?) |
|--------|--------------------------|-------------------------------|
| Heure de coucher reguliere 7j/7 | | |
| Chambre a ~17°C la nuit | | |
| Obscurite totale ou quasi-totale | | |
| Pas de cafeine apres 14h | | |
| Pas d'alcool pour "aider a dormir" | | |
| Pas d'ecrans lumineux 45-60 min avant coucher | | |
| Exercice physique regulier | | |

Puis :
1. Identifiez les **2 perturbateurs les plus actionnables** pour vous (pas les plus graves — les plus faciles a changer).
2. Formulez pour chacun une **micro-action concrete** : "A partir de [date], je vais [action specifique] — c'est faisable parce que [condition]."
3. Definissez comment vous allez le **mesurer** sur 2 semaines (ex. : journal de sommeil, note de qualite subjective).

### Criteres de reussite

- [ ] La grille des leviers est completee honnement pour chaque item
- [ ] Exactement 2 perturbateurs sont selectionnes avec une justification de leur actionnabilite (pas juste "c'est mauvais")
- [ ] Chaque micro-action est formulee avec : quoi, quand, condition de realisabilite
- [ ] Un indicateur de mesure est defini (pas "je verrai si ca va mieux" — une mesure concrete)
- [ ] Le plan n'est pas un programme complet de transformation — 2 changements maximum

## Exercice 3 — Decrire la CBT-I a quelqu'un qui ne la connait pas

### Objectif
Consolider la comprehension de la CBT-I (traitement de premiere ligne de l'insomnie chronique) en l'expliquant dans ses propres mots — technique Feynman.

### Consigne

Repondez par ecrit aux questions suivantes, comme si vous l'expliquiez a un ami qui vous dit : *"J'ai des problemes de sommeil depuis 6 mois, mon medecin m'a parle de CBT-I mais ca ne m'a pas l'air serieux comparé a un somnifere."*

1. **Qu'est-ce que la CBT-I** — en 3-4 phrases, sans jargon, avec une analogie si necessaire.
2. **Pourquoi l'ACP la recommande en premiere intention** avant les somniferes — citez au moins un chiffre de la recherche (module 02, ou source externe si vous l'avez consultee).
3. **En quoi la restriction du sommeil** (un composant CBT-I) semble paradoxale au premier abord — et pourquoi ca fonctionne mecaniquement.
4. **Une limite** : dans quel cas la CBT-I seule n'est pas suffisante et qu'est-ce qui doit etre evalue en complement ?

### Criteres de reussite

- [ ] La definition de la CBT-I en 3-4 phrases est claire, sans jargon, et correcte sur le fond
- [ ] Au moins un chiffre est cite (latence d'endormissement, eveil nocturne, ou duree des effets)
- [ ] La restriction du sommeil est expliquee mecaniquement (pourquoi reduire le temps au lit peut augmenter l'efficacite du sommeil)
- [ ] La limite citee est pertinente medicalement (ex. : apnees du sommeil, trouble psychiatrique comorbide, cause organique)
- [ ] Aucune affirmation medicale excessive ("la CBT-I guerit tout") — ton factuel et nuance
