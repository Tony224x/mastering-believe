## Exercice 1 — Planifier 150 minutes d'activite hebdomadaire

### Objectif
Passer de la recommandation OMS abstraite (150 min/sem) a un planning personnel concret et realiste, en tenant compte de contraintes reelles.

### Consigne

**Etape 1 — Inventaire de votre semaine**

Listez vos contraintes horaires reelles pour la semaine prochaine (travail, deplacement, famille, etc.). Identifiez les **creneaux disponibles** de 20 min ou plus.

**Etape 2 — Classification de vos activites actuelles**

Si vous pratiquez deja une activite physique, classifiez-la :

| Activite pratiquee | Frequence actuelle | Intensite (moderee/vigoureuse) | Minutes/sem |
|-------------------|-------------------|-------------------------------|------------|
| | | | |
| | | | |
| **Total actuel** | | | |

Rappel : 1 min vigoureuse = 2 min moderee pour le calcul OMS.

**Etape 3 — Construction du planning**

Construisez un planning sur 7 jours qui atteint **au minimum 150 min de cardio modere** ET **2 sessions de renforcement musculaire** :

| Jour | Activite prevue | Duree | Intensite | Type (cardio/force) |
|------|----------------|-------|----------|---------------------|
| Lundi | | | | |
| Mardi | | | | |
| Mercredi | | | | |
| Jeudi | | | | |
| Vendredi | | | | |
| Samedi | | | | |
| Dimanche | | | | |
| **Total cardio** | | | | |

**Etape 4 — Identifier les frictions**

Pour chaque session de renforcement prevue, identifiez :
- Ou (salle de sport, maison, exterieur) ?
- Avec quoi (poids du corps, halteres, machines) ?
- Principale friction anticipee (manque de temps, fatigue, motivation, autres) ?

### Criteres de reussite

- [ ] Le planning est concret : activites nommees (pas "sport"), jours et durees precises
- [ ] Le total de cardio modere atteint au minimum 150 min/sem
- [ ] 2 sessions de renforcement musculaire sont planifiees avec lieu et materiel notes
- [ ] Au moins une friction reelle est identifiee avec une solution anticipee concrete
- [ ] Le planning est **realisable** pour vous cette semaine, pas un ideal theorique

## Exercice 2 — Evaluer son niveau de sedentarite et definir des pauses

### Objectif
Mesurer concretement le temps passe assis et planifier des interruptions regulieres de la sedentarite.

### Consigne

**Etape 1 — Estimation du temps assis** (sur 3 jours consecutifs — incluant un jour de week-end)

Pour chaque jour, estimez le temps passe dans les categories suivantes :

| Categorie | Jour 1 (h) | Jour 2 (h) | Jour 3 (h) |
|-----------|-----------|-----------|-----------|
| Travail/etudes assis | | | |
| Transport assis | | | |
| Repas assis | | | |
| Ecrans (loisirs) assis | | | |
| **Total "temps assis"** | | | |
| Activite physique (debout/en mouvement) | | | |
| Sommeil | | | |

*Verification : Total temps assis + activite + sommeil ≈ 24h par jour.*

**Etape 2 — Identifier vos "blocs de sedentarite"**

Identifiez les blocs continus assis de plus de **60 minutes** dans votre journee type. Listez-les :

- Exemple : "9h-13h : travail bureau sans interruption = 4h continue"

**Etape 3 — Concevoir un systeme de pauses**

Pour chaque bloc identifie, concevez une interruption concrete :
- Que faire (5 min de marche, serie de squats, aller chercher de l'eau a l'autre bout du batiment, etc.)
- Declencheur (minuterie, fin de reunion, application, autre)
- Comment l'integrer sans perturber votre travail

### Criteres de reussite

- [ ] Le tableau est complete pour au moins 3 jours consecutifs (pas une estimation vague — un jour par jour)
- [ ] Les blocs de sedentarite continues > 60 min sont identifies et quantifies
- [ ] Au moins 2 strategies de pause sont concues avec un declencheur concret (pas "je vais essayer de me lever plus souvent")
- [ ] Vous avez calcule votre "ratio sedentarite/eveil" (temps assis / (24h - sommeil)) et l'avez interprete par rapport aux donnees OMS

## Exercice 3 — Lire et interpreter une etude sur l'exercice

### Objectif
Appliquer les outils du Module 01 a une source reelle sur l'activite physique — lire une meta-analyse avec esprit critique.

### Consigne

Lisez le resume suivant (extrait et simplifie de Shailendra P et al., 2022, *Am J Prev Med*) :

> *"Nous avons examine les associations entre le renforcement musculaire et la mortalite dans 16 cohortes prospectives. Les participants pratiquant toute quantite de renforcement musculaire presentaient un risque de mortalite toutes causes inferieur de 15 % (HR = 0,85, IC95 % [0,80-0,90]) par rapport a ceux qui n'en pratiquaient pas. La relation etait non lineaire : la reduction maximale de mortalite (~27 %) etait atteinte autour de 60 min/semaine de renforcement ; au-dela, les benefices additionnels etaient limites."*

Repondez aux questions suivantes :

1. **Type d'etude** : s'agit-il d'un ECR, d'une meta-analyse d'ECR, d'une meta-analyse de cohortes ? Quelle implication pour l'interpretation causale ?
2. **Lisez le HR** : HR = 0,85 [IC95 % 0,80-0,90]. Qu'est-ce que cela signifie en termes de risque relatif ? L'IC95 % inclut-il 1,0 ? Qu'est-ce que cela implique ?
3. **Relation non lineaire** : que signifie le fait que le benefice soit maximal a 60 min/sem et que pratiquer 3h/sem n'apporte pas 3x plus de benefices ?
4. **Facteurs confondants potentiels** : citez 2 biais ou confondants possibles dans une cohorte prospective sur l'exercice.
5. **Implication pratique** : si vous deviez formuler une recommandation pratique a partir de cette meta-analyse uniquement, quelle serait-elle — en restant honnete sur les limites ?

### Criteres de reussite

- [ ] Le type d'etude est correctement identifie (meta-analyse de cohortes) avec implication sur la causalite
- [ ] Le HR de 0,85 est interprete correctement (reduction de 15 % du risque relatif, IC non incluant 1,0 = significatif)
- [ ] La relation non lineaire est expliquee avec un argument logique (pas juste "plus c'est plus c'est moins")
- [ ] 2 facteurs confondants pertinents pour l'exercice sont proposes
- [ ] La recommandation pratique est formulee avec les reserves appropriees (niveau de preuve observationnel mentionné)
