## Notes anti-clivant (a respecter dans chaque module)

### Regles absolues

1. **Pas de guerres de regimes** : keto, vegan, carnivore ne sont jamais presentes comme "le meilleur regime". Le schema mediterraneen est un *exemple documente*, non une obligation.
2. **Jeune intermittent** : mentionne comme strategie possible de gestion calorique, sans promesse de longevite. Contre-indications mentionnees.
3. **Complements alimentaires** : uniquement via Examine.com (A→F). Le message par defaut est "alimentation > pilule". Preuve solide requise pour toute mention positive.
4. **Molecules de longevite off-label** (metformine, rapamycine, NMN, senolytiques) : *uniquement comme objets de recherche* avec statut d'essai explicite. Jamais comme recommandations.
5. **Vaccins & consensus medical** : alignement OMS/NIH/CDC. Tout contenu anti-vaccin est exclu.
6. **Walker *Why We Sleep*** : porte d'entree de vulgarisation uniquement. Les chiffres citables s'appuient sur AASM/SRS, pas sur le livre.
7. **Ton non-moralisateur** : exposer le risque (donnees), laisser la decision a l'apprenant + son medecin. Pas de culpabilisation sur le poids, l'alcool ou le tabac.

### Rappel disclaimer medical

Chaque module de theorie contient **au minimum un bloc disclaimer** (en encadre cite Markdown `>`).

## Scope du capstone (module 07)

Le capstone `07-capstone-plan-sante` produit :
- Un plan sante personnel sur **12 semaines**
- Minimum **3 habitudes** issues des modules precedents, formulees en objectifs SMART
- **Indicateurs mesurables** (heures de sommeil, minutes d'activite, etc.)
- Un **plan de suivi medical** (examens, biomarqueurs) a discuter avec un medecin
- Un **disclaimer explicite** : le plan est un point de depart educatif, pas une prescription

Le capstone ne vit pas dans `04-projects/` (pas de code), mais dans `01-theory/07-capstone-plan-sante.md` avec les exercices associes dans `03-exercises/`.

## Fil rouge pedagogique

**Le module 01 (esprit critique / niveau de preuve) cadre tout le reste.** Chaque module revient a la question : "Quel est le niveau de preuve ? Quelle est la taille d'effet ? Quelle incertitude reste-t-il ?"

Ce positionnement protege contre la hype, les guerres de regimes, et les biais de confirmation — tout en permettant d'enseigner un noyau comportemental robuste et actionnable.
