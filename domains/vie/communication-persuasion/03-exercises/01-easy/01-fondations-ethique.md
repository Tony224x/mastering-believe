## Exercice 1 — Identifier ethos, pathos, logos

### Objectif

Reconnaitre les trois dimensions du trio aristotelicien dans des messages reels et identifier ce qui manque lorsqu'un message ne convainc pas.

### Consigne

Lisez les trois messages ci-dessous. Pour chacun :
1. Identifiez quelles dimensions (ethos / pathos / logos) sont presentes et lesquelles sont absentes.
2. Expliquez en une phrase pourquoi le message risque d'etre peu convaincant.
3. Proposez une version revisee (3-5 phrases) qui integre les trois dimensions.

**Message A** — Email a un medecin traitant pour demander un arret de travail :
> "Bonjour Docteur, je suis vraiment epuise, je n'en peux plus, j'ai besoin d'un arret. S'il vous plait."

**Message B** — Proposition a un ami pour choisir un restaurant :
> "Les avis Google donnent 4,7/5 sur 340 avis, la carte comporte 12 options vegan, et le metro le plus proche est a 400 metres."

**Message C** — Demande a un chef de service pour obtenir une formation :
> "Je travaille dans ce service depuis 5 ans avec de bons resultats (ethos). Une formation en gestion de projet m'aiderait a livrer le dossier X dans les delais (logos). Je sais que le budget formation est contraint cette annee, donc je propose de faire cette formation en ligne, qui coute 3 fois moins (logos + prise en compte de son contrainte)."

### Criteres de reussite

- [ ] Les trois dimensions sont identifiees correctement pour chaque message (present / absent).
- [ ] L'explication de la faiblesse du message est precise (pas juste "il manque quelque chose").
- [ ] La version revisee de A et B integre les trois dimensions de facon naturelle, sans forcer.
- [ ] La version revisee reste dans les faits (pas d'invention de credentials faux ou de pathos exagere).

## Exercice 2 — Appliquer le filtre CTR

### Objectif

Evaluer si un message persuasif passe le filtre CTR (Consentement / Transparence / Reciprocite honnete) et identifier precisement quel garde-fou est enfreint.

### Consigne

Pour chacune des situations ci-dessous, evaluez si le message respecte le filtre CTR.
- Si OUI, expliquez en une phrase pourquoi.
- Si NON, indiquez quel(s) garde-fou(x) est/sont enfreint(s) et pourquoi.
- Pour les situations NON-CTR, proposez une version qui passerait le filtre.

**Situation 1** — Un vendeur de complement alimentaire ecrit :
> "Seulement 3 boites restent en stock ! Des milliers de clients ont perdu du poids avec notre produit. Ne ratez pas cette chance unique."

**Situation 2** — Un responsable a son equipe :
> "J'aimerais vous proposer de tester de nouvelles horaires variables sur deux mois. Vous etes libres de garder vos horaires actuels si vous preferez — je veux juste voir ce qui fonctionne mieux pour tout le monde. Votre avis compte pour la decision finale."

**Situation 3** — Un ami essaie de vous convaincre d'investir dans son projet :
> "J'ai besoin de 2 000 euros pour lancer mon projet. Je te rembourserai avec 10 % d'interets dans 6 mois." (il omet de mentionner qu'il a deja deux autres dettes en cours non remboursees)

### Criteres de reussite

- [ ] Chaque analyse identifie avec precision quel garde-fou CTR est respecte ou viole.
- [ ] L'analyse de la Situation 1 identifie au minimum deux violations (rarete artificielle = T, preuve sociale non verifiable = T).
- [ ] La Situation 2 est reconnue comme ethique.
- [ ] La version corrigee de la Situation 3 integre la transparence sur les faits materiels (les dettes en cours).

## Exercice 3 — Construire un message ethos/pathos/logos

### Objectif

Produire un message persuasif complet qui mobilise les trois dimensions et passe le filtre CTR — a partir d'une situation de la vie quotidienne.

### Consigne

Choisissez **une** des situations suivantes (celle qui vous parle le plus) et redigez un message de 100-150 mots qui integre explicitement les trois dimensions du trio aristotelicien.

**Situation A** — Vous souhaitez convaincre votre bailleur de faire reparer une infiltration que vous avez signalee il y a 3 semaines sans reponse.

**Situation B** — Vous souhaitez convaincre votre partenaire de reduire le temps d'ecran le soir pour favoriser le sommeil de toute la famille.

**Situation C** — Vous souhaitez convaincre votre responsable de vous laisser presenter votre travail lors d'une reunion client a laquelle vous n'etez habituellement pas invite.

Apres avoir redige le message, annotez-le : indiquez entre crochets [E] pour ethos, [P] pour pathos, [L] pour logos, a cote de chaque phrase ou segment concerne.

Verifiez ensuite que votre message passe le filtre CTR : une ligne par garde-fou (C / T / R).

### Criteres de reussite

- [ ] Les trois dimensions ethos, pathos, logos sont presentes et annotees dans le message.
- [ ] Le pathos illustre un enjeu reel (pas une peur exageree ou fabriquee).
- [ ] L'ethos s'appuie sur des faits verifiables, pas sur de l'autoproclamation.
- [ ] Le filtre CTR est verifie explicitement (une ligne par garde-fou).
- [ ] Le message est entre 100 et 150 mots.
