## 6. Role de l'IA dans ce systeme

> Specifie le mode, le timing et le prompt type — pas juste "j'utiliserai ChatGPT".

### Mode tuteur socratique
- **Quand :** (debut de chaque module ? Quand tu bloques ?)
- **Prompt type :** *"Je veux apprendre [concept]. Ne m'explique pas directement — pose-moi des questions pour tester ce que je sais deja, puis guide-moi par questions successives."*

### Mode generateur de retrieval
- **Quand :** (avant chaque session de revision ?)
- **Prompt type :** *"Genere [N] questions sur [themes], du plus simple au plus complexe. Ne donne pas les reponses — je reponds d'abord, tu corriges ensuite."*

### Mode partenaire Feynman
- **Quand :** (fin de chaque module ? 1 fois/semaine ?)
- **Prompt type :** *"Je vais expliquer [concept] comme si je l'enseignais. Detecte dans mon explication les zones floues, les sauts logiques, le jargon non defini et les erreurs conceptuelles."*

### Limites a maintenir
- [ ] Je ne lirai pas les explications passives du LLM sans retrieval actif juste apres
- [ ] Je verifierai les facts critiques dans les sources primaires (le LLM peut confabuler)
- [ ] Je fermerai la conversation et ferai un blank-page avant d'aller plus loin

## 7. Boucle metacognitive

### Bilan hebdomadaire (dimanche, 10 min)
- Score du test de la semaine : ___
- Ratio "je comprends quand je lis / je peux produire sans aide" : ___/10
- Ce qui a bien fonctionne cette semaine : ___
- Ce qui n'a pas fonctionne : ___
- Ajustement pour la semaine suivante : ___

### Declencheurs d'ajustement de strategie

| Signal | Action |
|--------|--------|
| Score < 60 % deux semaines de suite sur un type de concept | Reduire l'intervalle de revision + ajouter une session Feynman |
| Ratio "comprendre / produire" tres asymetrique apres 3 semaines | Ajouter des sessions de production active (ecriture, parler, coder) |
| Sensation de plateau prolonge | Identifier la faiblesse precise + cibler avec pratique deliberee |
| Plus aucune erreur sur un type d'exercice | Augmenter la difficulte / passer au niveau superieur |
