# Exercices — Module 02 : Retrieval practice

> **Niveau** : Débutant → Intermédiaire | **Temps estimé** : 40-50 min
> Ces exercices supposent que tu as lu le module 02 et fait les exercices du module 01.

## Exercice 1 — Blank-page recall structuré sur le module 02

### Objectif
Pratiquer le blank-page recall sur le module 02 et évaluer ta restitution avec une grille de critères explicites.

### Consigne
1. Ferme le fichier `02-retrieval-practice.md`.
2. Sur feuille blanche, réponds aux 5 questions suivantes **de mémoire, sans regarder** :
   - a. Nomme et décris les **4 techniques concrètes** présentées dans le module.
   - b. Cite les **deux chiffres de rappel** de Roediger & Karpicke (2006) à 1 semaine (groupes SSSS vs STTT).
   - c. Explique en 2-3 phrases pourquoi l'échec pendant un test renforce quand même la mémoire.
   - d. Qu'est-ce que l'"effet de génération" ? (auteur + définition)
   - e. Explique pourquoi enseigner à un pair est une forme de retrieval practice — pas juste une activité pédagogique.
3. Rouvre le module et note ton score : 1 point si la réponse est correcte et complète, 0,5 si partielle, 0 si absente ou incorrecte.
4. Score maximal : 5 points. Note tes lacunes et planifie de les re-tester demain.

### Critères de réussite
- [ ] Les 5 questions traitées (même si la réponse est incomplète)
- [ ] Score >= 3/5 (si < 3, c'est normal — note les lacunes et reviens demain)
- [ ] Les lacunes identifiées sont notées pour révision espacée (J+1)
- [ ] Tu n'as pas regardé le module avant d'avoir répondu aux 5 questions

## Exercice 2 — Reformuler 3 énoncés de flashcards

### Objectif
Apprendre à distinguer une bonne flashcard (qui force la récupération) d'une mauvaise (qui autorise la reconnaissance).

### Consigne
Voici 3 flashcards mal formulées. Pour chacune, explique le problème, puis reformule-la en version efficace.

**Flashcard A (mal formulée)**
- Recto : "Qu'est-ce que le retrieval practice ?"
- Verso : "C'est quand on se teste."

**Flashcard B (mal formulée)**
- Recto : "L'étude de Karpicke & Roediger (2008) sur le vocabulaire swahili montre que..."
- Verso : "...se tester est mieux que re-étudier."

**Flashcard C (mal formulée)**
- Recto : "Enseigner à un pair"
- Verso : "C'est utile pour consolider l'apprentissage."

Pour chaque reformulation, vérifie que :
- La question est précise et complète
- La réponse contient au moins un chiffre, un mécanisme ou une nuance qui distingue un vrai connaisseur d'un ignorant

### Critères de réussite
- [ ] 3 problèmes identifiés (pas juste "c'est vague" — expliquer *pourquoi* c'est problématique)
- [ ] 3 reformulations produites
- [ ] Chaque reformulation contient un élément quantitatif ou mécaniste dans la question ou la réponse
- [ ] Les reformulations sont testées : quelqu'un qui ne connaît pas le sujet ne pourrait pas deviner la réponse

## Exercice 3 — Session retrieval practice sur un sujet réel + debriefing "pair"

### Objectif
Appliquer le protocole complet (blank-page + enseignement) à un contenu que tu dois vraiment apprendre, puis utiliser la variante "enseigner" pour détecter les zones floues restantes.

### Consigne
Choisis un sujet que tu dois apprendre ou réviser cette semaine (cours, compétence pro, lecture, autre domaine de ce repo...).

**Phase 1 — Étude et recall (35 min)**

1. **Étude initiale (15 min)** : lis/étudie le contenu une seule fois, en annotant les points clés.
2. **Blank-page recall (10 min)** : ferme tout, écris tout ce dont tu te souviens.
3. **Vérification (5 min)** : compare avec la source, note les lacunes avec `[LACUNE]`.
4. **Flashcards (5 min)** : crée 3 flashcards sur les lacunes identifiées.

**Phase 2 — Enseignement à un pair (10-15 min)**

5. Explique le sujet à voix haute à quelqu'un (ou à toi-même, à une chaise vide) comme si tu étais le professeur. Dure 5-7 min.
6. Note **exactement** les moments où tu as buté ou utilisé des formules vagues ("en gros...", "je crois que...") — ce sont tes zones floues.
7. **Bilan écrit (5 min)** : en 5-8 lignes, réponds à ces trois questions :
   - Quelles zones floues l'enseignement a-t-il révélées que le blank-page recall n'avait pas détectées ?
   - As-tu ressenti l'illusion de compétence à un moment ? Quand ?
   - Quelles flashcards supplémentaires tu créerais maintenant ?

### Critères de réussite
- [ ] Un sujet réel choisi (pas un exercice inventé)
- [ ] Les 4 phases de la Phase 1 réalisées dans l'ordre sans skipper
- [ ] L'explication orale réalisée (seul ou à un pair) pendant au moins 5 min
- [ ] Au moins 2 zones floues identifiées pendant l'enseignement
- [ ] Bilan écrit de 5-8 lignes incluant les trois réponses demandées
