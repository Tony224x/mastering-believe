# Exercices — Module 02 : Retrieval practice

> **Niveau** : Debutant | **Temps estime** : 35-45 min
> Ces exercices supposent que tu as lu le module 02 et fait les exercices du module 01.

## Exercice 1 — Active recall structure sur le module 02

### Objectif
Pratiquer le blank-page recall sur le module 02 et evaluer ta restitution avec une grille de criteres explicites.

### Consigne
1. Ferme le fichier `02-retrieval-practice.md`.
2. Sur feuille blanche, reponds aux 5 questions suivantes **de memoire, sans regarder** :
   - a. Nomme et decris les 3 techniques concretes presentees dans le module.
   - b. Quel est le chiffre de rappel a 1 semaine dans Karpicke & Roediger (2008) pour le "meilleur" groupe ?
   - c. Explique en 2-3 phrases pourquoi l'echec pendant un test renforce quand meme la memoire.
   - d. Qu'est-ce que l'"effet de generation" ?
   - e. Decris le protocole d'une session d'etude de 45 min integrant le retrieval practice.
3. Rouvre le module et note ton score : 1 point si la reponse est correcte et complete, 0.5 si partielle, 0 si absente ou incorrecte.
4. Score maximal : 5 points. Note tes lacunes et planifie de les re-tester demain.

### Criteres de reussite
- [ ] Les 5 questions traitees (meme si la reponse est incomplete)
- [ ] Score >= 3/5 (si < 3, c'est normal — note les lacunes et reviens demain)
- [ ] Les lacunes identifiees sont notees pour revision espacee (J+1)
- [ ] Tu n'as pas regarde le module avant d'avoir repondu aux 5 questions

## Exercice 2 — Reformuler 3 enonces de flashcards

### Objectif
Apprendre a distinguer une bonne flashcard (qui force la recuperation) d'une mauvaise (qui autorise la reconnaissance).

### Consigne
Voici 3 flashcards mal formulees. Pour chacune, explique le probleme, puis reformule-la en version efficace.

**Flashcard A (mal formulee)**
- Recto : "Qu'est-ce que le retrieval practice ?"
- Verso : "C'est quand on se teste."

**Flashcard B (mal formulee)**
- Recto : "L'etude de Karpicke & Roediger (2008) sur le vocabulaire swahili montre que..."
- Verso : "...se tester est mieux que re-etudier."

**Flashcard C (mal formulee)**
- Recto : "Le blank-page recall"
- Verso : "On ferme ses documents et on ecrit ce qu'on se souvient."

Pour chaque reformulation, verifie que :
- La question est precise et complete
- La reponse contient au moins un chiffre, un mecanisme ou une nuance qui distingue un vrai connaisseur d'un ignorant

### Criteres de reussite
- [ ] 3 problemes identifies (pas juste "c'est vague" — expliquer *pourquoi* c'est problematique)
- [ ] 3 reformulations produites
- [ ] Chaque reformulation contient un element quantitatif ou mecaniste dans la question ou la reponse
- [ ] Les reformulations sont testees : quelqu'un qui ne connait pas le sujet ne pourrait pas deviner la reponse

## Exercice 3 — Planifier une session d'etude pour un sujet reel

### Objectif
Appliquer le protocole retrieval practice a un contenu que tu dois vraiment apprendre (pas d'exercice fictif).

### Consigne
Choisis un sujet que tu dois apprendre ou reviser cette semaine (cours, competence pro, lecture, autre domaine de ce repo...).

1. **Etude initiale (20 min)** : lis/etudie le contenu une seule fois, en annotant les points cles.
2. **Blank-page recall (10 min)** : ferme tout, ecris tout ce dont tu te souviens.
3. **Verification (5 min)** : compare avec la source, note les lacunes avec `[LACUNE]`.
4. **Flashcards (10 min)** : cree 5 flashcards sur les points les plus importants ou les lacunes identifiees.
5. **Bilan ecrit** : en 5-8 lignes, decris ce que tu as remarque pendant l'exercice — qu'est-ce qui etait plus difficile que prevu ? Ou l'illusion de competence s'est-elle manifestee ?

### Criteres de reussite
- [ ] Un sujet reel choisi (pas un exercice inventé)
- [ ] Les 4 phases realisees dans l'ordre sans skipper
- [ ] Au moins 3 lacunes identifiees dans la verification
- [ ] 5 flashcards creees (respectant les regles de formulation)
- [ ] Bilan ecrit de 5-8 lignes incluant au moins une reference au concept d'"illusion de competence"
