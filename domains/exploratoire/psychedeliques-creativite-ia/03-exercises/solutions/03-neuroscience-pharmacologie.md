# Solutions — Module 03 : Neuroscience & pharmacologie

La psilocybine est un objet de recherche clinique sous protocole controle avec screening strict, pas une pratique d'auto-experimentation.

## Solution Exercice 1 — Classer les affirmations par niveau de preuve

1. **Etabli** — l'agonisme 5-HT2A comme mediateur principal est solidement documente (bloquer le recepteur attenue fortement les effets ; confirmation par imagerie d'occupation chez l'humain).
2. **Speculatif** — REBUS est un cadre theorique influent, mais la **causalite** (relaxation des priors → benefice therapeutique) n'est **pas demontree** chez l'humain.
3. **Etabli / replique** — la baisse de connectivite interne du DMN sous psilocybine est repliquee (revue systematique), malgre une heterogeneite methodologique.
4. **Speculatif / preliminaire** — la neuroplasticite est **etablie en preclinique** (rongeurs/drosophile) ; son equivalence chez l'humain n'est pas confirmee a la meme granularite.
5. **Preliminaire** — la phase 3 a atteint son endpoint primaire, mais le **delta clinique est modeste** et les donnees completes sont en attente de peer-review ; ce n'est **pas** un traitement disponible.

**Idee cle** : la competence centrale du module est de distinguer **etabli / preliminaire / speculatif**, et de reperer quand un discours public presente du speculatif comme de l'etabli.

## Solution Exercice 2 — Reperer les biais methodologiques

Au moins trois limites attendues (idealement les quatre) :

1. **Auto-selection** : l'annonce "participez a une etude sur les psychedeliques" attire des volontaires aux attentes positives → biais d'echantillon, effets surestimes.
2. **Desaveuglage** : les effets sont si perceptibles que les participants devinent leur groupe → l'**expectancy** gonfle les resultats, le double aveugle est compromis.
3. **Petit effectif (n=30)** : faible generalisabilite et incapacite a detecter les effets indesirables rares.
4. **Biais de publication** : "aucune mention d'un resultat nul d'un sous-groupe" suggere une publication selective des resultats positifs.
5. **Suivi court (4 semaines)** : pas de donnees sur le maintien a long terme.

**Effet global sur l'interpretation** : ces biais convergent vers une **surestimation** de l'effet et une **faible generalisabilite**. Conclusion honnete : un tel resultat est, au mieux, un signal a confirmer — pas une preuve.

## Solution Exercice 3 — Observation vs theorie

1. **Observation** — "la variabilite (entropie) du signal augmente" est une mesure d'imagerie.
2. **Theorie/interpretation** — "l'entropie est la cause de la richesse de la conscience" est un cadre theorique debattu, **a ne pas presenter comme un fait**.
   *Reformulation* : "Selon l'hypothese de l'entropie cerebrale (debattue), la variabilite accrue du signal pourrait etre liee a la qualite des etats de conscience."
3. **Observation** — "baisse de connectivite interne du DMN" est une mesure repliquee.
4. **Sur-interpretation** — une correlation d'imagerie **ne prouve pas** une guerison.
   *Reformulation* : "On observe une baisse de connectivite du DMN ; son lien avec un eventuel benefice clinique reste interpretatif et non demontre causalement."

**Idee cle** : decrire une observation ≠ prouver un mecanisme ≠ prouver un benefice. Beaucoup de discours sautent ces etapes — c'est exactement ce que le module apprend a reperer. Aucune de ces reponses n'introduit de dosage ni de conseil d'usage.
