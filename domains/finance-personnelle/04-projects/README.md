# Gabarit de plan financier personnel — Finance Personnelle

> ⚠️ **Document éducatif uniquement.** Ce gabarit est un outil pédagogique issu du domaine "Finance Personnelle" (mastering-believe). Il ne constitue pas un conseil financier, fiscal ou en investissement personnalisé. Les projections reposent sur des hypothèses qui ne garantissent pas les résultats futurs. Consultez un conseiller agréé pour toute décision réelle. Tout investissement comporte un risque de perte en capital.

---

## Comment utiliser ce gabarit

1. Faites une copie de ce fichier dans votre dossier `03-exercises/workspace/` (espace gitignore — vos données restent locales)
2. Remplissez chaque section avec vos données réelles
3. Utilisez le simulateur `02-code/07-capstone-plan-financier.py` pour calculer votre projection
4. Révisez ce document une fois par an
